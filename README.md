🧠 Synapse – Multidisciplinary Technical Club Website

Synapse is the official website of the Multidisciplinary Technical Club, designed to represent innovation, collaboration, and technical excellence across multiple emerging domains. The platform serves as a digital hub for students to explore the club’s vision, domains, events, and team in a modern, interactive, and user-friendly way.

The website features a futuristic glassmorphism UI, smooth animations, and a responsive layout optimized for all devices. It highlights the club’s core domains including Computer Science, Artificial Intelligence & Machine Learning, Data Science, Cyber Security, and Internet of Things (IoT).

Key sections include an engaging landing page, domain showcases, upcoming and past events, a dynamic team page with expandable roles, and quick access to club resources through integrated links. The navigation system supports smooth scrolling and route-based transitions for a seamless user experience.

Built using modern web technologies, Synapse focuses on performance, accessibility, and clean component-based architecture, making it scalable and easy to maintain.

🚀 Tech Stack

React + TypeScript

Tailwind CSS

React Router

Lucide Icons

🎯 Purpose

To create a professional digital identity for the Synapse Technical Club and provide students with a centralized platform for learning, collaboration, and innovation.
